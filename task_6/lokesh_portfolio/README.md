# Lokesh Balaga Professional Portfolio

This is a professional portfolio site built with Flask, featuring my skills as a Computer Science student and developer.

## Features

- Modern landing page highlighting projects, education, and experience
- Individual project details pages
- Responsive design and professional aesthetics
- Contact form for direct communication

## Running the Project

1. Install Flask:
    ```
    pip install flask
    ```
2. Run the app:
    ```
    python app.py
    ```
3. Open in your browser: [http://127.0.0.1:5000/](http://127.0.0.1:5000/)
