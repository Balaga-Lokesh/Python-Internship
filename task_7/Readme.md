# Image Resizer Tool

## How it works
- Resizes all images in the 'images' folder to 256x256 pixels and saves to 'resized_images' as PNG.
- Supports common formats: .jpg, .png, etc.

## Usage
- Put your images in 'images/'.
- Edit size and format in the script as needed.
- Run: python resize_script.py

## Requirements
- Python 3.x, Pillow
